Introducción a Flexbox y CSS Grid

Flexbox y CSS Grid son los motores modernos de maquetación en CSS. Flexbox resuelve alineaciones en una sola dimensión (fila o columna) y es ideal para componentes UI como menús y tarjetas, mientras que Grid organiza elementos en dos dimensiones (filas y columnas) para construir layouts completos y responsivos.

Conceptos clave y diferencias

CaracterísticaFlexboxCSS Grid
DimensiónUnidimensionalBidimensional
FlujoOrden natural de elementosEstructura explícita con plantilla
Alineación cruzadaSí (ejes principal y cruzado)Sí (filas, columnas y áreas)
Casos de uso típicosBarras de navegación, galeríasLayouts de páginas, dashboards
Soporte superposiciónLimitadoSofisticado (grid-area)

Flexbox a fondo

1. Activar Flexbox

.flex-container { display: flex; }

El contenedor padre se convierte en flex-container y los hijos en flex-items.

2. Propiedades del contenedor

PropiedadDescripciónValores comunes
flex-directionDefine el eje principalrow, column, row-reverse, column-reverse
flex-wrapPermite varias filasnowrap, wrap, wrap-reverse
justify-contentDistribuye elementosflex-start, center, space-between, space-around, space-evenly
align-itemsAlinea en eje cruzadostretch, center, flex-start, flex-end, baseline
gapEspacio entre ítemscualquier unidad (e.g. 1rem)

3. Propiedades de ítems

  • order: cambia el orden visual sin alterar HTML.
  • flex-grow: distribuye espacio sobrante.
  • flex-shrink: controla contracción.
  • flex-basis: tamaño inicial.
  • align-self: alineación individual.

4. Ejercicio guiado: mini-navbar

.topbar { display: flex; justify-content: space-between; align-items: center; padding: 1rem; background: #0f2027; }

@media (max-width: 600px) { .topbar { flex-direction: column; } }

CSS Grid paso a paso

1. Activar Grid

.grid-demo { display: grid; }

2. Definir filas y columnas

.grid-demo { grid-template-columns: repeat(3, 1fr); grid-template-rows: auto 200px; gap: 1rem; }

3. Posicionar ítems

.item-a { grid-column: 2 / 4; grid-row: 1 / span 2; }

Usa grid-template-areas para nombrar zonas y simplificar el layout.

Buenas prácticas

  • Usa gap en lugar de margin para separar ítems.
  • Combina Flexbox con min-height y auto-margins para centrar verticalmente.
  • Empieza mobile-first: inicia con una columna y expande con media queries.
  • Degrada con @supports not (display:grid) para navegadores antiguos.
